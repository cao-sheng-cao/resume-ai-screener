使用说明 v1.0.2

本版本新增模型选择和 Token 消耗显示。

上传 GitHub：
1. 把 package.json、main.js、src、.github、README.md 上传到仓库根目录。
2. 进入 Actions。
3. 运行 Build Windows Installer。
4. 成功后下载 Artifacts 里的 Resume-AI-Screener-Windows-Installer。

使用应用：
1. 打开软件。
2. 输入 DeepSeek API Key。
3. 选择模型：快速模式适合批量筛选，推理模式适合重点候选人复核。
4. 设置岗位标准。
5. 上传简历。
6. 点击评分。
7. 查看评分结果、置信度、证据、Token 消耗和排行榜。

不要上传或公开你的 API Key。
